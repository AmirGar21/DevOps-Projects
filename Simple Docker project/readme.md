# Учебный проект по Docker


## Part 1. Готовый Docker

Скачать и запустить официальный образ nginx.

Проверить запуск, замапленные порты и IP.

Перезапустить контейнер и проверить доступность стартовой страницы.

## Part 2. Операции с контейнером

Прочитать конфиг nginx внутри контейнера.

Настроить отдельный конфиг на локальной машине для страницы /status.

Скопировать конфиг в контейнер, перезапустить nginx и проверить /status.

Экспортировать контейнер в файл, удалить образ и контейнер, затем импортировать обратно.

Проверить, что /status всё ещё работает.

## Part 3. Мини веб-сервер

Написать простой сервер на C + FastCGI, возвращающий "Hello World!".

Настроить nginx проксировать запросы с порта 81 на сервер на 8080.

Проверить работу страницы.

Сохранить конфиг nginx для дальнейшего использования.

## Part 4. Свой Docker

Написать Dockerfile для сборки мини-сервера и запуска nginx.

Копировать конфиг nginx внутрь образа.

Собрать образ, запустить с маппингом портов и локальной папки с конфигом.

Проверить работу сервера и страницы /status.

## Part 5. Dockle

Просканировать созданный образ на безопасность.

Исправить ошибки и предупреждения.

## Part 6. Docker Compose

Написать docker-compose.yml для двух контейнеров:

Мини-сервер из Part 5.

Nginx-прокси, который перенаправляет запросы на первый контейнер.

Настроить маппинг порта 8080 второго контейнера на 80 локальной машины.

Собрать и запустить проект через docker-compose.

Проверить, что страницы работают через браузер.
